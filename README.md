## âš¡ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Sundargopaldas/Lumini-I.A.git
   cd Lumini-I.A
   ```

2. **Setup Backend**
   ```bash
   cd backend
   npm install
   # Configure .env file if needed
   npm start
   ```

3. **Setup Frontend**
   ```bash
   # Open a new terminal
   cd frontend
   npm install
   npm run dev
   ```

4. **Access the App**
   Open [http://localhost:5173](http://localhost:5173) in your browser.
