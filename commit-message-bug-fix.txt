fix: Corrigir bug ao deletar transacoes no Dashboard

Problema:
- Ao tentar deletar uma transacao diretamente do Dashboard, ocorria um erro
- O componente TransactionCard tentava chamar onDelete() mas a prop nao era passada no Dashboard

Solucao:
- Adicionado CustomAlert ao Dashboard para confirmacoes
- Implementado handleDeleteTransaction com logica completa de delecao
- Atualizado TransactionCard no Dashboard para receber prop onDelete
- Adicionado setLoading(true) no inicio do fetchTransactions
- Agora os usuarios podem deletar transacoes tanto do Dashboard quanto da pagina de Transacoes

Arquivos modificados:
- frontend/src/pages/Dashboard.jsx: Importacao CustomAlert, estado de alerta, handleDeleteTransaction, onDelete prop
